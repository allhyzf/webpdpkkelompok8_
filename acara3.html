ACARA 3:

VISUALISASI DATA CTD MENGGUNAKAN OCEAN DATA VIEW

Acara 3 berjudul "Visualisasi Data CTD Menggunakan Ocean Data View" dan merupakan bagian dari Praktikum Pengolahan Data Perikanan dan Kelautan. Modul ini memandu mahasiswa untuk memvisualisasikan data CTD (Conductivity, Temperature, Depth) menggunakan perangkat lunak Ocean Data View (ODV). Proses visualisasi meliputi langkah-langkah persiapan data, seperti mengedit data di Excel dan Notepad++, serta mengatur parameter dalam ODV agar data dapat divisualisasikan dengan tepat. Bertujuan untuk membantu mahasiswa memahami cara memvisualisasikan parameter oseanografi dan mempermudah analisis data kelautan.
